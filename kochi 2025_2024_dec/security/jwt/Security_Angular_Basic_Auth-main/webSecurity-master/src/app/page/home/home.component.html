Bem-vindo ao Sistema de Estudo de Guards no Angular

Este sistema foi desenvolvido para demonstrar o uso de Guards no Angular. Guards permitem controlar o acesso a diferentes partes de uma aplicação baseado em regras de autenticação e autorização.

Funcionalidades do Sistema:

  • Autenticação: Implementação de Guards para controlar se o usuário está autenticado antes de acessar certas rotas.
  • Autorização: Verificação de permissões específicas para acessar determinados recursos, como roles (admin, usuário comum).

Utilize o menu acima para navegar entre as funcionalidades. Explore os exemplos práticos de autenticação e autorização usando Guards do Angular.
